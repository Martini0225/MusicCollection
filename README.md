# MusicCollection
音樂庫管理

左邊為歌曲清單，可用滑鼠或鍵盤上下方向鍵選取歌曲，右側會顯示歌曲資訊如歌手/音樂家名稱、專輯名稱與年份、圖片與歌詞，若歌曲沒歌詞則不會顯示

接著是右側左下角的鉛筆圖示，選擇左邊清單一首歌曲，再點選此鉛筆圖示，就會開啟編輯歌曲資訊的表單(和新增歌曲表單是一樣的)，歌手/音樂家、歌曲名稱以及YouTube連結也都一定要填寫的，如果這三個空白或不符合格式，系統會提醒並還原成更改前的內容
若沒有從左側清單選取歌曲就點選鉛筆圖示，系統會提醒你要先從左側清單選取一首歌來編輯

最後是右側左下角的垃圾桶圖示，選擇左邊清單一首歌曲，再點選此垃圾桶圖示，系統會向你確認是否將這首歌從資料庫永久刪除，按下「是」，就會成功刪除，按下「否」則不會刪除。
若沒有從左側清單選取歌曲就點選垃圾桶圖示，系統會提醒你要先從左側清單選取一首歌來刪除
