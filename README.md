# MusicCollection
音樂庫管理

左邊為歌曲清單，可用滑鼠或鍵盤上下方向鍵選取歌曲，右側會顯示歌曲資訊如歌手/音樂家名稱、專輯名稱與年份、圖片與歌詞，若歌曲沒歌詞則不會顯示
